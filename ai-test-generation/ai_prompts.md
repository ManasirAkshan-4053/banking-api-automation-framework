# AI Test Generation Prompts

This folder demonstrates how AI can assist in generating automation tests for UI applications.

## Example Prompt

Generate Playwright Python test for login page validation using PyTest.

Test steps:

1. Open login page
2. Enter username and password
3. Click login button
4. Verify dashboard page loads successfully

## Purpose

This approach helps accelerate automation development by using AI to generate initial test scripts which can then be refined by QA engineers.

## Technologies Used

* Playwright (UI automation)
* PyTest (test framework)
* AI-assisted test generation
